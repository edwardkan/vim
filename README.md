# vim

Install vim

1) git Clone 
git clone https://github.com/edwardkan/vim.git

2) mv vim .vim

3) Create symbolic links so that ~/.vimrc points to the ~/.vim/vimrc file: 
ln -s ~/.vim/vimrc ~/.vimrc

-----------------------------------

Install tmux 

1) sudo apt-get install tmux

2) ln -s ~/.vim/tmux.conf ~/.tmux.conf
